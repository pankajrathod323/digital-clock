# Digital Clock

A simple digital clock web application built with JavaScript, HTML, and CSS.

## Features

- Real-time display of current time
- 12-hour or 24-hour format (customizable)
- Responsive and minimal design

## Getting Started

1. Clone the repository:
  ```bash
  git clone https://github.com/your-username/digital-clock.git
  ```
2. Open `index.html` in your browser.

## Usage

- The clock updates automatically every second.
- Customize the appearance by editing the CSS.
